# Check House CRM

CRM متخصص للمطابخ والتصميم الداخلي - شركة Check House

## الميزات الأساسية

### 1. إدارة بيانات العميل
- أبعاد المطبخ (طول × عرض × ارتفاع)
- نوع الاختيارة (بولي لاك/HPL/أكريليك)
- نوع الأجهزة المطلوبة
- فيديو موقع المشروع

### 2. مفيش عروض المنتجات
- عرض الأسعار بالمتر
- حساب تلقائي (عدد أثاث × سعر المتر)
- تقديم العرض للعملاء

### 3. متابعة الدفعات
- تسجيل (مقدم + تسليم + ضمان)
- تنبيهات للتأخير

### 4. مرحلة القياس والمعاينة
- تسجيل زيارات الموقع
- رفع صور وملاحظات المعاينة

### 5. ضمان ما بعد التسليم
- تاريخ انتهاء الضمان
- طلبات الشراء من العملاء القدامى

## البدء السريع

```bash
npm install
npm run dev
```
